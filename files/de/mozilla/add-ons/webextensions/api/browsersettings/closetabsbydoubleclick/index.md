Ein {{WebExtAPIRef("types.BrowserSetting", "BrowserSetting")}}-Objekt, das verwendet werden kann, um die Fähigkeit des Benutzers, einen Tab mit einem Doppelklick zu schließen, zu aktivieren oder zu deaktivieren.

Der zugrunde liegende Wert ist ein Boolean.

Standardmäßig ist `closeTabsByDoubleClick` auf false gesetzt. Die Einstellung kann vom Benutzer in about:config geändert werden.
