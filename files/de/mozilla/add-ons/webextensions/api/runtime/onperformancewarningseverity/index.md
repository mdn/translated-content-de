Der Schweregrad der Warnung, die das {{WebExtAPIRef("runtime.onPerformanceWarning")}}-Ereignis ausgelöst hat.

## Typ

Werte dieses Typs sind Zeichenketten. Mögliche Werte sind:

- `"low"`
- `"medium"`
- `"high"`
