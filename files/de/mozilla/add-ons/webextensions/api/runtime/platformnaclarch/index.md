{{AddonSidebar}}

Die Native Client-Architektur. Diese kann sich auf einigen Plattformen von `arch` unterscheiden.

## Typ

Werte dieses Typs sind Zeichenfolgen. Mögliche Werte sind: `"arm"`, `"x86-32"`, `"x86-64"`.
