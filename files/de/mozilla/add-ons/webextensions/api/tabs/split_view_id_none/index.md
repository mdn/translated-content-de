Ein spezieller ID-Wert, der Tabs zugewiesen wird, die sich nicht in einer [geteilten Ansicht](/de/docs/Mozilla/Add-ons/WebExtensions/Working_with_the_Tabs_API#working_with_tab_split_views) befinden.

Sein Wert ist `-1`.
