WebAssembly SIMD-Bitweise-Anweisungen.

- [`shl`](/de/docs/WebAssembly/Reference/SIMD/bitwise/shl)
  - : Arithmetische Linksverschiebung. Verschiebt die Bits in jedem Lane einer [`v128`](/de/docs/WebAssembly/Reference/Types/v128)-Wertinterpretation um den gleichen angegebenen Betrag nach links.
- [`shr_s`](/de/docs/WebAssembly/Reference/SIMD/bitwise/shr_s)
  - : Arithmetische Rechtsverschiebung. Verschiebt die Bits in jedem Lane einer `v128`-Wertinterpretation um den gleichen angegebenen Betrag nach rechts und gibt signierte Werte aus.
- [`shr_u`](/de/docs/WebAssembly/Reference/SIMD/bitwise/shr_u)
  - : Logische Rechtsverschiebung. Verschiebt die Bits in jedem Lane einer `v128`-Wertinterpretation um den gleichen angegebenen Betrag nach rechts und gibt unsignierte Werte aus.
