Dieser Abschnitt dokumentiert die verschiedenen WebAssembly-Typen.

## Referenztypen

- [`externref`](/de/docs/WebAssembly/Reference/Types/externref)
  - : Referenziert einen JavaScript-Wert, sodass er einem Wasm-Modul übergeben werden kann, ohne dass Kopieren oder Serialisieren erforderlich ist.
- [`funcref`](/de/docs/WebAssembly/Reference/Types/funcref)
  - : Referenziert eine in Wasm definierte Funktion, sodass höherwertige Funktionen über die Grenzen der Wasm- und JavaScript-Sprachen hinweg verwendet werden können.
