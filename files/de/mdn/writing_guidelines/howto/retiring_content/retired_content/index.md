Diese Seite listet Abschnitte der MDN Web Docs auf, die ausgemustert wurden. Jeder Eintrag enthält einen Link zu den archivierten Inhalten im [MDN Museum Repository](https://github.com/mdn/museum) und zum GitHub-Issue oder zur Diskussion, in der die Entscheidung zur Ausmusterung getroffen wurde.

Einzelheiten darüber, wie die Ausmusterung funktioniert, finden Sie im [Leitfaden zur Ausmusterung von Inhalten](/de/docs/MDN/Writing_guidelines/Howto/Retiring_content).

## Ausgemusterte Abschnitte

### Canvas Raycaster

- **Ausgemustert:** Juni 2022
- **Archivierte Inhalte:** [museum/canvas-raycaster](https://github.com/mdn/museum/tree/main/canvas-raycaster)
- **Issue auf GitHub:** [Canvas-raycaster in mdn/museum verschieben](https://github.com/mdn/mdn/issues/193)

Eine Canvas API Raycaster-Demo und die dazugehörige Dokumentationsseite. Die Inhalte wurden im Rahmen der Repository-Wartung ins Museum verschoben.

> [!NOTE]
> Diese Ausmusterung erfolgte vor dem [Leitfaden zur Ausmusterung von Inhalten](/de/docs/MDN/Writing_guidelines/Howto/Retiring_content) Prozess. Es gibt keine verbleibende Diskussion.

### WebVR

- **Ausgemustert:** September 2022
- **Archiv im GitHub:** [museum/webvr](https://github.com/mdn/museum/tree/main/webvr)
- **PR auf GitHub:** [WebVR-Tests importieren](https://github.com/mdn/museum/pull/6)

Eine Sammlung von WebVR API-Testbeispielen, einschließlich grundlegender Display-Informationen, Bühnenparametern, roher WebGL, VR-Controller-Demos und einer A-Frame-Demo. Die WebVR API wurde zugunsten der [WebXR Device API](/de/docs/Web/API/WebXR_Device_API) veraltet.

> [!NOTE]
> Diese Ausmusterung erfolgte vor dem [Leitfaden zur Ausmusterung von Inhalten](/de/docs/MDN/Writing_guidelines/Howto/Retiring_content) Prozess. Es gibt keine verbleibende Diskussion.
