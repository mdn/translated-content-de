## Grundlagen der Gestaltung 1

Unsere grundlegende Bewertungsaufgabe zur Formular-Gestaltung ist recht freizügig, und Sie haben viel Flexibilität, was Sie letzten Endes machen. Aber Ihr CSS sollte versuchen, die folgenden Anforderungen zu erfüllen:

1. Fügen Sie eine Art von leichtem "Reset" hinzu, um Schriftarten, Abstände, Ränder und Größen von Anfang an konsistenter zu machen.
2. Darüber hinaus fügen Sie eine schöne, konsistente Gestaltung für die Eingaben und den Button hinzu.
3. Verwenden Sie eine Art Layout-Technik, um Eingaben und Bezeichnungen ordentlich auszurichten.
