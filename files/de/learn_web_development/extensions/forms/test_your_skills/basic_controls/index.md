Das Ziel dieses Fertigkeitstests ist es, zu prüfen, ob Sie unseren Artikel [Grundlegende native Formularsteuerelemente](/de/docs/Learn_web_development/Extensions/Forms/Basic_native_form_controls) verstanden haben.

> Wenn Sie nicht weiterkommen, können Sie sich an uns über einen unserer [Kommunikationskanäle](/de/docs/MDN/Community/Communication_channels) wenden.

## Grundlegende Steuerungen 1

Diese Aufgabe beginnt sanft und bittet Sie, zwei `<input>`-Elemente für die Benutzer-ID und das Passwort sowie eine Schaltfläche zum Absenden zu erstellen.

1. Erstellen Sie geeignete Eingabefelder für Benutzer-ID und Passwort.
2. Sie sollten diese auch semantisch mit ihren Textetiketten verknüpfen.
3. Erstellen Sie eine Absenden-Schaltfläche innerhalb des verbleibenden Listenpunkts mit dem Schaltflächentext "Log in".

## Grundlegende Steuerungen 2

In der nächsten Aufgabe müssen Sie funktionierende Gruppen von Kontrollkästchen und Optionsschaltflächen aus den vorgegebenen Textetiketten erstellen.

1. Wandeln Sie den Inhalt des ersten `<fieldset>` in eine Gruppe von Optionsschaltflächen um - Sie sollten nur einen Pony-Charakter gleichzeitig auswählen können.
2. Stellen Sie sicher, dass das erste Optionsfeld beim Laden der Seite ausgewählt ist.
3. Wandeln Sie den Inhalt des zweiten `<fieldset>` in eine Gruppe von Kontrollkästchen um.
4. Fügen Sie ein paar weitere Hotdog-Optionen hinzu.

## Grundlegende Steuerungen 3

Die letzte Aufgabe in diesem Set erfordert, dass Sie einen Dateiauswahl-Dialog erstellen.

1. Erstellen Sie einen einfachen Dateiauswahl-Dialog.
2. Erlauben Sie dem Benutzer, mehrere Dateien gleichzeitig auszuwählen.
3. Der Dateiauswahl-Dialog soll nur JPG- und PNG-Bilder akzeptieren.
