HTTP wurde erstmals in den frühen 1990er Jahren spezifiziert. Mit Blick auf Erweiterbarkeit entworfen, erfuhr es im Laufe der Jahre zahlreiche Ergänzungen; dies führte dazu, dass seine Spezifikation auf zahlreiche Spezifikationsdokumente verteilt wurde (inmitten experimenteller, aufgegebener Erweiterungen). Diese Seite listet relevante Ressourcen zu HTTP auf.
