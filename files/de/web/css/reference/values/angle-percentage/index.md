Der **`<angle-percentage>`** [CSS](/de/docs/Web/CSS) [Datentyp](/de/docs/Web/CSS/Reference/Values/Data_types) repräsentiert einen Wert, der entweder ein {{Cssxref("angle")}} oder ein {{Cssxref("percentage")}} sein kann.

Wenn ein `<angle-percentage>` als zulässiger Typ angegeben ist, bedeutet dies, dass der Prozentsatz in einen Winkel aufgelöst wird und daher in einem {{cssxref("calc()")}}-Ausdruck verwendet werden kann.

## Syntax

Für Details zu den einzelnen Syntaxen, die durch diesen Typ erlaubt sind, konsultieren Sie die Dokumentation für {{Cssxref("angle")}} und {{Cssxref("percentage")}}.
