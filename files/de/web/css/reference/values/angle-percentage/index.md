Der **`<angle-percentage>`** [CSS](/de/docs/Web/CSS) [Datentyp](/de/docs/Web/CSS/Reference/Values/Data_types) repräsentiert einen Wert, der entweder ein {{Cssxref("angle")}} oder ein {{Cssxref("percentage")}} sein kann.

Wenn ein `<angle-percentage>` als erlaubter Typ angegeben ist, bedeutet dies, dass der Prozentwert in einen Winkel aufgelöst wird und daher in einem {{Cssxref("calc", "calc()")}} Ausdruck verwendet werden kann.

## Syntax

Verweisen Sie auf die Dokumentation für {{Cssxref("angle")}} und {{Cssxref("percentage")}}, um Details zu den einzelnen syntaktischen Möglichkeiten dieses Typs zu erhalten.
