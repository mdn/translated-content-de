**CSS-Werte** sind die Werte, die CSS-Eigenschaften zugewiesen werden können. Diese umfassen Schlüsselwörter, [Datentypen](/de/docs/Web/CSS/Reference/Values/Data_types) und [Funktionen](/de/docs/Web/CSS/Reference/Values/Functions).

> [!NOTE]
> Diese Seite ist ein Index aller Wert-Features in CSS. Die Seite [CSS-Werte und -Einheiten](/de/docs/Web/CSS/Guides/Values_and_units) führt in das Modul ein, das einige, aber nicht alle dieser Werte definiert.
