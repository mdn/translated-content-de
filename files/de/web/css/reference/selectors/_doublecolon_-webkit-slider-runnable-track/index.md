Das **`::-webkit-slider-runnable-track`** [CSS](/de/docs/Web/CSS) [Pseudoelement](/de/docs/Web/CSS/Reference/Selectors/Pseudo-elements) repräsentiert den "Track" (die Rille, in der der Indikator gleitet) eines {{HTMLElement("input/range", '&lt;input type="range"&gt;')}}.

## Syntax

```css
::-webkit-slider-runnable-track {
  /* ... */
}
```

## Spezifikationen

Nicht Teil eines Standards.
