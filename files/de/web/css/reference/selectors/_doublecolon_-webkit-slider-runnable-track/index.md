{{non-standard_header}}

Das **`::-webkit-slider-runnable-track`** [CSS](/de/docs/Web/CSS) [Pseudo-Element](/de/docs/Web/CSS/Reference/Selectors/Pseudo-elements) repräsentiert die "Schiene" (die Vertiefung, in der der Indikator gleitet) eines {{HTMLElement("input/range", '&lt;input type="range"&gt;')}}.

## Syntax

```css
::-webkit-slider-runnable-track {
  /* ... */
}
```

## Spezifikationen

Teil keiner Norm.
