Das **CSS Farbanpassungsmodul** bietet ein Modell und steuert die automatische Farbanpassung durch den Benutzeragenten, um Benutzerpräferenzen wie "Dunkelmodus", Kontrasteinstellung und andere Farbschemavorgaben zu berücksichtigen.

Zusammen mit den {{CSSxRef("@media")}} Features {{cssxref("@media/prefers-color-scheme", "prefers-color-scheme")}}, {{cssxref("@media/prefers-contrast", "prefers-contrast")}} und {{cssxref("@media/forced-colors", "forced-colors")}} definiert dieses Modul, wie und wann Farben automatisch vom Browser angepasst werden.
