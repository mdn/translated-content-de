Das **CSS Inline-Layout** Modul definiert die Block-Achsen-Ausrichtung und Größenbestimmung von Inhalten auf Inline-Ebene und fügt einen speziellen Layout-Modus für Initialen hinzu. Es beschreibt das CSS-Formatierungsmodell für einen Fluss von Elementen und Text innerhalb eines Containers, der über mehrere Zeilen umbrochen wird.
