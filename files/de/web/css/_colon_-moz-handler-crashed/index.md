{{Non-standard_header}}

Die **`:-moz-handler-crashed`** [CSS](/de/docs/Web/CSS) [Pseudoklasse](/de/docs/Web/CSS/Pseudo-classes) ist eine [Mozilla-Erweiterung](/de/docs/Web/CSS/Reference/Mozilla_extensions), die auf Elemente angewendet wird, die nicht angezeigt werden können, weil das Plugin, das für ihre Darstellung verantwortlich ist, abgestürzt ist.

> [!NOTE]
> Dieser Selektor ist hauptsächlich für die Verwendung durch Theme-Entwickler vorgesehen.

## Syntax

```css
:-moz-handler-crashed {
  /* ... */
}
```

## Spezifikationen

Ist kein Teil eines Standards.
