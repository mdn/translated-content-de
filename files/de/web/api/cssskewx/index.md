{{APIRef("CSS Typed OM")}}{{AvailableInWorkers}}

Die **`CSSSkewX`**-Schnittstelle der [CSS Typed Object Model API](/de/docs/Web/API/CSS_Object_Model) repräsentiert den [`skewX()`](/de/docs/Web/CSS/Reference/Values/transform-function/skewX)-Wert der individuellen {{CSSXRef('transform')}}-Eigenschaft in CSS. Sie erbt Eigenschaften und Methoden von ihrem Elternteil [`CSSTransformValue`](/de/docs/Web/API/CSSTransformValue).

{{InheritanceDiagram}}

## Konstruktor

- [`CSSSkewX()`](/de/docs/Web/API/CSSSkewX/CSSSkewX)
  - : Erstellt ein neues `CSSSkewX`-Objekt.

## Instanz-Eigenschaften

- [`ax`](/de/docs/Web/API/CSSSkewX/ax)
  - : Gibt den Wert der x-Achse zurück oder legt ihn fest.
