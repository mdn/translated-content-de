{{APIRef("CSSOM")}}

Die schreibgeschützte **`types`**-Eigenschaft der [`CSSViewTransitionRule`](/de/docs/Web/API/CSSViewTransitionRule)-Schnittstelle gibt ein Array zurück, das die `types`-Deskriptorwerte der zugehörigen {{cssxref("@view-transition")}}-At-Regel enthält.

## Wert

Ein Array von Strings.

Wenn kein `types`-Deskriptor für die `@view-transition`-At-Regel festgelegt ist, entspricht die `types`-Eigenschaft einem leeren Array.

## Beispiele

Siehe die Hauptseite der [`CSSViewTransitionRule`](/de/docs/Web/API/CSSViewTransitionRule) für Anwendungsbeispiele.
