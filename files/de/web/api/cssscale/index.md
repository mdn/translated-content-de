{{APIRef("CSS Typed Object Model API")}}

Das **`CSSScale`** Interface der [CSS Typed Object Model API](/de/docs/Web/API/CSS_Object_Model) repräsentiert die [scale()](/de/docs/Web/CSS/Reference/Values/transform-function/scale) und [scale3d()](/de/docs/Web/CSS/Reference/Values/transform-function/scale3d) Werte der individuellen {{CSSXRef('transform')}}-Eigenschaft in CSS. Es erbt Eigenschaften und Methoden von seinem Elternteil [`CSSTransformValue`](/de/docs/Web/API/CSSTransformValue).

{{InheritanceDiagram}}

## Konstruktor

- [`CSSScale()`](/de/docs/Web/API/CSSScale/CSSScale)
  - : Erstellt ein neues `CSSScale` Objekt.

## Instanzeigenschaften

- [`x`](/de/docs/Web/API/CSSScale/x)
  - : Gibt den Wert der x-Achse zurück oder setzt diesen.
- [`y`](/de/docs/Web/API/CSSScale/y)
  - : Gibt den Wert der y-Achse zurück oder setzt diesen.
- [`z`](/de/docs/Web/API/CSSScale/z)
  - : Gibt den Wert der z-Achse zurück oder setzt diesen.
