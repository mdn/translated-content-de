Die **`hand`**-Eigenschaft des [`Gamepad`](/de/docs/Web/API/Gamepad)-Interfaces gibt ein Enum zurück, das angibt, in welcher Hand der Controller gehalten wird oder höchstwahrscheinlich gehalten werden sollte.

## Wert

Ein [`GamepadHand`](https://w3c.github.io/gamepad/extensions.html#gamepadhand-enum)-Enum; mögliche Werte sind:

- `left` — die linke Hand.
- `right` — die rechte Hand.
- `""` (Leerer String) – wenn die anderen Werte nicht zutreffen. Z.B. wenn der Controller in beiden Händen gehalten wird oder in beiden Händen gleichermaßen geeignet wäre.

## Beispiele

Noch nicht festgelegt
