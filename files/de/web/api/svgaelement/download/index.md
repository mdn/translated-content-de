{{APIRef("SVG")}}

Die **`download`**-Eigenschaft der [`SVGAElement`](/de/docs/Web/API/SVGAElement)-Schnittstelle gibt einen String zurück, der anzeigt, dass der Browser die verlinkte URL als Download behandeln soll.

Diese Eigenschaft kann gesetzt werden, um den `download`-Wert der URL zu ändern. Sie spiegelt den Wert des {{svgattr("download")}}-Attributs wider.

> [!NOTE]
> Dieser Wert wird möglicherweise nicht für den Download verwendet. Dieser Wert kann nicht verwendet werden, um zu bestimmen, ob der Download stattfinden wird.

## Wert

Ein String.
