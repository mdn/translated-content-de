{{APIRef("Canvas API")}}

Die **`CanvasRenderingContext2D.miterLimit`**-Eigenschaft der Canvas 2D API legt das Miter-Limit-Verhältnis fest.

> [!NOTE]
> Weitere Informationen über Miters finden Sie unter [Anwenden von Stilen und Farben](/de/docs/Web/API/Canvas_API/Tutorial/Applying_styles_and_colors) im [Canvas-Leitfaden](/de/docs/Web/API/Canvas_API/Tutorial).

## Wert

Eine Zahl, die das Miter-Limit-Verhältnis in Koordinatenraumeinheiten angibt. Werte wie Null, negativ, {{jsxref("Infinity")}} und {{jsxref("NaN")}} werden ignoriert. Der Standardwert ist `10.0`.

### Verwendung der `miterLimit`-Eigenschaft
