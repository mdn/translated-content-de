Der **`CSSMathMax()`**-Konstruktor erstellt ein neues [`CSSMathMax`](/de/docs/Web/API/CSSMathMax)-Objekt, das die CSS-Funktion {{CSSXref('max', 'max()')}} repräsentiert.

## Syntax

```js-nolint
new CSSMathMax(args)
```

### Parameter

- `args`
  - : Eine Liste von Zahlen oder [`CSSNumericValue`](/de/docs/Web/API/CSSNumericValue)-Objekten.

### Ausnahmen

- [`TypeError`](/de/docs/Web/JavaScript/Reference/Global_Objects/TypeError)
  - : Wird ausgelöst, wenn ein _Fehler_ beim Hinzufügen aller Werte in `args` auftritt.
