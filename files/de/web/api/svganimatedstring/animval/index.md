Die `animVal`-Eigenschaft der [`SVGAnimatedString`](/de/docs/Web/API/SVGAnimatedString)-Schnittstelle ist eine schreibgeschützte Zeichenfolge, die den animierten Wert des reflektierten Attributs darstellt.

## Wert

Als Zeichenfolge.

Wenn das angegebene Attribut oder die Eigenschaft animiert wird, ist dies der aktuelle animierte Wert.
Wenn das angegebene Attribut oder die Eigenschaft derzeit nicht animiert wird, entspricht dieser Wert dem Wert der [`SVGAnimatedString.baseVal`](/de/docs/Web/API/SVGAnimatedString/baseVal)-Eigenschaft.
