{{APIRef("WebRTC")}}

Die **`totalSamplesReceived`**-Eigenschaft des [`RTCInboundRtpStreamStats`](/de/docs/Web/API/RTCInboundRtpStreamStats)-Wörterbuchs gibt die Gesamtanzahl der in diesem Stream empfangenen Samples an, einschließlich [`concealedSamples`](/de/docs/Web/API/RTCInboundRtpStreamStats/concealedSamples).

> [!NOTE]
> Der Wert ist für Videostreams undefiniert.

## Wert

Eine positive ganze Zahl.
