Die schreibgeschützte Eigenschaft **`TimeEvent.view`** identifiziert das {{Glossary("WindowProxy", "WindowProxy")}}, von dem das Ereignis erzeugt wurde.

## Wert

Ein {{Glossary("WindowProxy", "WindowProxy")}}-Objekt, das das Fenster identifiziert, das das Ereignis erzeugt hat, oder `null`, wenn kein Fenster zugeordnet ist.
