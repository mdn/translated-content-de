Die **`value`** schreibgeschützte Eigenschaft der [`PerformanceTimingConfidence`](/de/docs/Web/API/PerformanceTimingConfidence)-Schnittstelle ist ein allgemeines Vertrauensmaß dafür, ob ein Leistungsdatensatz die typische Anwendungsleistung widerspiegelt oder wahrscheinlich durch externe Faktoren beeinflusst wird.

## Wert

Ein enumerierter Wert von `low` oder `high`, der jeweils geringes oder hohes Vertrauen anzeigt.

Siehe die Hauptseite zu [`PerformanceTimingConfidence`](/de/docs/Web/API/PerformanceTimingConfidence) für ein Beispiel.
