Der **`XSLTProcessor()`** Konstruktor erstellt eine neue [`XSLTProcessor`](/de/docs/Web/API/XSLTProcessor) Objektinstanz.

## Syntax

```js-nolint
new XSLTProcessor()
```

### Parameter

Keine.

### Rückgabewert

Eine neue [`XSLTProcessor`](/de/docs/Web/API/XSLTProcessor) Objektinstanz.
