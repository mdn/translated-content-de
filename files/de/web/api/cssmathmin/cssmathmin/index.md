Der **`CSSMathMin()`** Konstruktor erzeugt ein neues [`CSSMathMin`](/de/docs/Web/API/CSSMathMin) Objekt, das die CSS {{CSSXref('min','min()')}} Funktion darstellt.

## Syntax

```js-nolint
new CSSMathMin(args)
```

### Parameter

- `args`
  - : Eine Liste von Zahlen oder [`CSSNumericValue`](/de/docs/Web/API/CSSNumericValue) Objekten.

### Ausnahmen

- [`TypeError`](/de/docs/Web/JavaScript/Reference/Global_Objects/TypeError)
  - : Wird ausgelöst, wenn es ein _Fehlschlag_ beim Hinzufügen aller Werte in `args` gibt.
