Die **`featurePolicy`**-Eigenschaft der [`Document`](/de/docs/Web/API/Document)-Schnittstelle ist eine schreibgeschützte Eigenschaft, die die Schnittstelle [`FeaturePolicy`](/de/docs/Web/API/FeaturePolicy) zurückgibt. Diese stellt eine einfache API zur Verfügung, um die [Berechtigungsrichtlinien](/de/docs/Web/HTTP/Guides/Permissions_Policy) zu überprüfen, die auf ein spezifisches Dokument angewendet werden.

## Wert

Ein [`FeaturePolicy`](/de/docs/Web/API/FeaturePolicy)-Objekt, das verwendet werden kann, um die auf das Dokument angewendeten Berechtigungsrichtlinieneinstellungen zu überprüfen.
