{{APIRef("HTML DOM")}}

Die **`form`**-Nur-Lese-Eigenschaft der [`HTMLSelectElement`](/de/docs/Web/API/HTMLSelectElement)-Schnittstelle gibt ein [`HTMLFormElement`](/de/docs/Web/API/HTMLFormElement)-Objekt zurück, das dieses {{htmlelement("select")}} besitzt, oder `null`, wenn dieses select nicht zu einem Formular gehört.

## Wert

Ein [`HTMLFormElement`](/de/docs/Web/API/HTMLFormElement) oder `null`.
