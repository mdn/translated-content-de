Die **`toJSON()`** Methode des [`PaymentAddress`](/de/docs/Web/API/PaymentAddress) Interfaces ist ein standardmäßiger Serializer, der eine JSON-Darstellung der Eigenschaften des PaymentAddress-Objekts zurückgibt.

## Syntax

```js-nolint
toJSON()
```

### Parameter

Keine.

### Rückgabewert

Ein JSON-Objekt.
