Die schreibgeschützte **`featurePolicy`**-Eigenschaft des [`HTMLIFrameElement`](/de/docs/Web/API/HTMLIFrameElement)-Interfaces gibt das [`FeaturePolicy`](/de/docs/Web/API/FeaturePolicy)-Interface zurück, das eine einfache API zum Untersuchen der auf einen bestimmten Frame angewendeten [Permissions Policies](/de/docs/Web/HTTP/Guides/Permissions_Policy) bereitstellt.

## Wert

Ein [`FeaturePolicy`](/de/docs/Web/API/FeaturePolicy)-Objekt, das verwendet werden kann, um die auf den Frame angewendeten Permissions Policy-Einstellungen zu untersuchen.
