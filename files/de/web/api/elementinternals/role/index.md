Die **`role`**-Eigenschaft der [`ElementInternals`](/de/docs/Web/API/ElementInternals)-Schnittstelle gibt die [WAI-ARIA-Rolle](/de/docs/Web/Accessibility/ARIA/Reference/Roles) des Elements zurück. Zum Beispiel könnte ein Auswahlkästchen [`role="checkbox"`](/de/docs/Web/Accessibility/ARIA/Reference/Roles/checkbox_role) haben. Sie spiegelt das `role`-Attribut wider und gibt die implizite ARIA-Rolle des Elements nur dann zurück, wenn sie explizit gesetzt wurde.

## Wert

Ein String, der eine ARIA-Rolle enthält. Eine vollständige Liste der ARIA-Rollen finden Sie auf der [ARIA-Techniken-Seite](/de/docs/Web/Accessibility/ARIA/Guides/Techniques).
