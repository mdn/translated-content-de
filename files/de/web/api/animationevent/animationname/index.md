{{APIRef("Web Animations")}}

Die schreibgeschützte **`AnimationEvent.animationName`**-Eigenschaft ist ein String, der den Wert der CSS-Eigenschaft {{cssxref("animation-name")}} enthält, die mit der Übergang verbunden ist.

## Wert

Ein String, der den Wert der CSS-Eigenschaft {{cssxref("animation-name")}} enthält.
