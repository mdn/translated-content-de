Die schreibgeschützte Eigenschaft **`hasPosition`** des [`GamepadPose`](/de/docs/Web/API/GamepadPose)-Interfaces gibt einen booleschen Wert zurück, der angibt, ob das [`Gamepad`](/de/docs/Web/API/Gamepad) die Möglichkeit hat, Positionsinformationen zu verfolgen und zurückzugeben.

## Wert

Ein boolescher Wert.
