Die **`usernameFragment`**-Eigenschaft des [`RTCIceCandidateStats`](/de/docs/Web/API/RTCIceCandidateStats)-Interfaces ist ein String, der das {{Glossary("ICE", "ICE")}}-Benutzernamenfragment ("ice-ufrag") darstellt.

Das Fragment identifiziert eindeutig eine einzelne ICE-Interaktionssitzung und kann verwendet werden, um Kommunikationsvorgänge zu identifizieren, die Teil derselben Sitzung sind, selbst bei ICE-Neustarts.

Für weitere Informationen siehe [`RTCIceCandidate.usernameFragment`](/de/docs/Web/API/RTCIceCandidate/usernameFragment).

## Wert

Ein String, der den [`usernameFragment`](/de/docs/Web/API/RTCIceCandidate/usernameFragment) des entsprechenden [`RTCIceCandidate`](/de/docs/Web/API/RTCIceCandidate) darstellt.
