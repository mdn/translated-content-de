Die **`candidateType`**-Eigenschaft des [`RTCIceCandidateStats`](/de/docs/Web/API/RTCIceCandidateStats)-Wörterbuchs ist ein String, der den Typ des {{Glossary("ICE", "ICE")}}-Kandidaten angibt, den das Objekt darstellt.

## Wert

Ein String, dessen Wert einer der Strings ist, die in [`RTCIceCandidate.type`](/de/docs/Web/API/RTCIceCandidate/type#value) gefunden werden.
