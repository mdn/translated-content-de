Die **`beta`**-Eigenschaft des [`DeviceMotionEventRotationRate`](/de/docs/Web/API/DeviceMotionEventRotationRate)-Interfaces, die nur lesbar ist, gibt die Rotationsrate um die X-Achse in Grad pro Sekunde an.

## Wert

Ein `double`, das die Rotationsrate um die X-Achse in Grad pro Sekunde angibt. Siehe [Erläuterungen zu Beschleunigungsmesserwerten](/de/docs/Web/API/Device_orientation_events/Detecting_device_orientation#accelerometer_values_explained) für Details.
