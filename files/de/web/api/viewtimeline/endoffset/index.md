{{APIRef("Web Animations")}}

Die schreibgeschützte Eigenschaft **`endOffset`** des [`ViewTimeline`](/de/docs/Web/API/ViewTimeline)-Interfaces gibt einen [`CSSNumericValue`](/de/docs/Web/API/CSSNumericValue) zurück, der die End- (100% Fortschritt) Scrollposition der Timeline als einen Offset vom Anfang des überfließenden Inhaltsbereichs im Scroller darstellt.

## Wert

Ein [`CSSNumericValue`](/de/docs/Web/API/CSSNumericValue).

## Beispiele

Siehe die Hauptseite von [`ScrollTimeline`](/de/docs/Web/API/ScrollTimeline) für ein Beispiel.
