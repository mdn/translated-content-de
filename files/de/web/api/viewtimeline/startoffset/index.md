{{APIRef("Web Animations")}}

Die schreibgeschützte Eigenschaft **`startOffset`** der [`ViewTimeline`](/de/docs/Web/API/ViewTimeline)-Schnittstelle gibt einen [`CSSNumericValue`](/de/docs/Web/API/CSSNumericValue) zurück, der die Start-Scroll-Position (0% Fortschritt) der Timeline als einen Offset vom Beginn des überfüllten Inhaltsbereichs im Scroller darstellt.

## Wert

Ein [`CSSNumericValue`](/de/docs/Web/API/CSSNumericValue).

## Beispiele

Sehen Sie sich die Hauptseite der [`ScrollTimeline`](/de/docs/Web/API/ScrollTimeline) für ein Beispiel an.
