Die schreibgeschützte **`crashReport`**-Eigenschaft der [`Window`](/de/docs/Web/API/Window)-Schnittstelle gibt ein [`CrashReportContext`](/de/docs/Web/API/CrashReportContext)-Objekt zurück, das es ermöglicht, beliebige Daten für den aktuellen obersten Browsing-Kontext aufzuzeichnen. Diese Daten werden dann in [`CrashReport`](/de/docs/Web/API/CrashReport)-Objekte aufgenommen, die an einen Reporting-Endpunkt gesendet werden, wenn ein Browser-Absturz auftritt.

## Wert

Eine Instanz eines [`CrashReportContext`](/de/docs/Web/API/CrashReportContext)-Objekts.
