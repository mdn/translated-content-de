Die **`crashReport`**-Schreibgeschützte Eigenschaft des [`Window`](/de/docs/Web/API/Window)-Interfaces gibt ein [`CrashReportContext`](/de/docs/Web/API/CrashReportContext)-Objekt zurück, das es ermöglicht, beliebige Daten für den aktuellen top-level Browsing-Kontext aufzuzeichnen. Diese Daten werden dann in [`CrashReport`](/de/docs/Web/API/CrashReport)-Objekte aufgenommen, die an einen Reporting-Endpunkt gesendet werden, wenn ein Browser-Absturz auftritt.

## Wert

Eine Instanz des [`CrashReportContext`](/de/docs/Web/API/CrashReportContext)-Objekts.
