Die statische Methode **`console.groupEnd()`** beendet die aktuelle Inline-Gruppe in der Konsole. Siehe [Verwendung von Gruppen in der Konsole](/de/docs/Web/API/console#using_groups_in_the_console) in der [`console`](/de/docs/Web/API/console) Dokumentation für Details und Beispiele.

## Syntax

```js-nolint
console.groupEnd()
```

### Parameter

Keine.

### Rückgabewert

Keiner ({{jsxref("undefined")}}).
