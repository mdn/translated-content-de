{{APIRef("CSSOM view API")}}

Die schreibgeschützte **`pageLeft`**-Eigenschaft des [`VisualViewport`](/de/docs/Web/API/VisualViewport)-Interfaces gibt die x-Koordinate des linken Rands des visuellen Viewports relativ zum Ursprung des initialen umschließenden Blocks in CSS-Pixeln zurück, oder `0`, wenn das aktuelle Dokument nicht vollständig aktiv ist.

## Wert

Ein double.
