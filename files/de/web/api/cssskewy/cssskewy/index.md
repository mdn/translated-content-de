Der **`CSSSkewY()`**-Konstruktor erstellt ein neues [`CSSSkewY`](/de/docs/Web/API/CSSSkewY)-Objekt, das den [`skewY()`](/de/docs/Web/CSS/Reference/Values/transform-function/skewY)-Wert der individuellen {{CSSXRef('transform')}}-Eigenschaft in CSS darstellt.

## Syntax

```js-nolint
new CSSSkewY(ay)
```

### Parameter

- [`ay`](/de/docs/Web/API/CSSSkewY/ay)
  - : Ein Wert für den `ay`-Winkel des zu konstruierenden [`CSSSkewY`](/de/docs/Web/API/CSSSkewY)-Objekts. Dies muss ein [`CSSNumericValue`](/de/docs/Web/API/CSSNumericValue) sein.
