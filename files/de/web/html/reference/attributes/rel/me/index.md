Das **`me`** Schlüsselwort für das [`rel`](/de/docs/Web/HTML/Reference/Elements/link#rel) Attribut der {{HTMLElement("link")}} und {{HTMLElement("a")}} Elemente zeigt an, dass die aktuelle Ressource von der verlinkten Partei repräsentiert wird. Der `me` Wert wurde in der [XHTML Friends Network (XFN)-Spezifikation](https://gmpg.org/xfn/) eingeführt.

```html
<link rel="me" href="example.com" />
```

Das `rel="me"` Attribut wird in den Spezifikationen [RelMeAuth](https://microformats.org/wiki/RelMeAuth) und [Web sign in](https://microformats.org/wiki/web-sign-in) verwendet, um einer Person die Identifikation bei einem Webdienst mittels ihres Domainnamens oder einer bestimmten URL zu ermöglichen.

## Browser-Kompatibilität

Browser-Kompatibilität ist irrelevant, da `rel="me"` von Webdiensten und nicht von Browsern verwendet wird.
