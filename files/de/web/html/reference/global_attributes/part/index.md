Das **`part`**-[globale Attribut](/de/docs/Web/HTML/Reference/Global_attributes) enthält eine durch Leerzeichen getrennte Liste der Part-Namen des Elements. Part-Namen ermöglichen es CSS, spezifische Elemente in einem Shadow-Dom über das {{CSSxRef("::part")}}-Pseudoelement auszuwählen und zu stylen.

Siehe das [`exportparts`-Attribut](/de/docs/Web/HTML/Reference/Global_attributes/exportparts) für ein Anwendungsbeispiel.
