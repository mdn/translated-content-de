Das WebDriver BiDi-Protokoll ist in Module unterteilt.
Jedes Modul repräsentiert eine Sammlung verwandter Befehle und Ereignisse, die in spezifischen Browser-Automatisierungsfällen verwendet werden.

Sowohl Befehls- als auch Ereignisnamen verwenden den Modulnamen als Präfix: `module_name.command_name` für Befehle und `module_name.event_name` für Ereignisse.
