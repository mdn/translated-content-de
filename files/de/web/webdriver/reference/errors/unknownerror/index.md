Der **unbekannte Element** Fehler ist ein [WebDriver-Fehler](/de/docs/Web/WebDriver/Reference/Errors), der auftritt, wenn ein nicht spezifizierter Fehler im Treiber auftritt, während ein [Befehl](/de/docs/Web/WebDriver/Reference/Classic/Commands) verarbeitet wird.

Ein unbekannter Fehler ist normalerweise spezifisch für einen bestimmten Treiber, daher ist es ratsam, die Fehlermeldung, falls vorhanden, zu lesen, um einen Hinweis darauf zu erhalten, was passiert ist.
