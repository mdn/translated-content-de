WebDriver Classic ist das ursprüngliche WebDriver-Protokoll, das HTTP-basierte Befehle für die Browserautomation bereitstellt. Diese Referenz dokumentiert die in WebDriver Classic verfügbaren Befehle, Fähigkeiten, Fehler und Zeitüberschreitungen.

- [Befehle](/de/docs/Web/WebDriver/Reference/Classic/Commands)
  - : HTTP-basierte Operationen zur Inspektion und Steuerung des Verhaltens von Browsern, Verwaltung von Browsersitzungen und Fenstern sowie zur Interaktion mit DOM-Elementen.
- [Fähigkeiten](/de/docs/Web/WebDriver/Reference/Classic/Capabilities)
  - : Konfigurationsoptionen, die verwendet werden, um die von einer Sitzung unterstützten Funktionen zu kommunizieren und zu definieren, welche Funktionen der Treiber bei der Erstellung einer neuen Sitzung erfüllen muss.
- [Fehler](/de/docs/Web/WebDriver/Reference/Classic/Errors)
  - : Fehlerantworten, die Befehle zurückgeben können, einschließlich HTTP-Statuscodes und JSON-Fehlernutzdaten.
- [Zeitüberschreitungen](/de/docs/Web/WebDriver/Reference/Classic/Timeouts)
  - : Definitionen von Zeitüberschreitungen, die das Verhalten für Skriptinjektion, Dokumentnavigation und Elementabfrage steuern.
