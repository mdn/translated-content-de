Eine **registrierfähige Domain** ist eine Domain, die von einer Einzelperson oder einer Organisation über einen Domain-Namen-Registrar registriert werden kann (oder bereits registriert wurde). Sie ist die höchste Ebene einer einzelnen {{Glossary("site", "Website")}} im Web.

Registrierfähige Domains werden direkt unter einer _effektiven Top-Level-Domain_ (eTLD) erstellt, wie zum Beispiel `.org`, `.com` oder `.ac.uk`. Daher wird eine registrierfähige Domain manchmal als "eTLD+1" bezeichnet.

Zum Beispiel sind alle folgenden registrierfähige Domains:

```plain
crookedtimber.org
theguardian.com
sussex.ac.uk
```

Alle Domains unter einer registrierfähigen Domain gehören zur selben Organisation. Zum Beispiel:

```plain
 film.theguardian.com
music.theguardian.com
```

```plain
      news.sussex.ac.uk
      blog.sussex.ac.uk
admissions.sussex.ac.uk
```

Beachten Sie, dass nicht alle eTLDs Top-Level-Domains sind, da viele Registrare es Organisationen erlauben, Domains auch unterhalb der obersten Ebene zu registrieren. Im obigen Beispiel ist `.ac.uk` eine Subdomain der Top-Level-Domain `.uk`, sodass `sussex.ac.uk` und `aber.ac.uk` an unterschiedliche Organisationen registriert werden können.

Da dies eine Frage der Richtlinien des Registrars ist, ist es algorithmisch unmöglich festzustellen, ob ein bestimmtes Domainnamensuffix (wie `.ac.uk`) öffentlich registrierbar ist oder nicht. Die [Public Suffix List](https://publicsuffix.org/) ist eine Liste aller Suffixe, unter denen Organisationen direkt Namen registrieren können: Das heißt, es ist eine Liste von eTLDs.
