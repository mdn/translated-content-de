**Internet Explorer** (oder IE) war ein kostenloser, grafischer {{Glossary("Browser", "Webbrowser")}}, der von Microsoft für den Einsatz in Unternehmensumgebungen verwaltet wurde. {{Glossary("Microsoft_Edge", "Microsoft Edge")}} ist derzeit der Standardbrowser unter Windows.

Microsoft integrierte IE erstmals 1995 in Windows als Teil des Pakets "Microsoft Plus!". Bis ungefähr 2002 war Internet Explorer der meistgenutzte Browser weltweit, verlor jedoch an Boden gegenüber {{Glossary("Google_Chrome", "Chrome")}}, {{Glossary("Mozilla_Firefox", "Firefox")}}, {{Glossary("Microsoft_Edge", "Edge")}} und {{Glossary("Apple_Safari", "Safari")}}.

IE durchlief viele Versionen und bot Versionen für Desktop, Mobilgeräte und die Xbox-Konsole. Er war auch auf Mac und UNIX verfügbar; Microsoft stellte diese Versionen 2003 bzw. 2001 ein. Die letzte Windows-Version war Version 11.0.220 vom 10. November 2020.

Microsoft beendete den Support für IE am 15. Juni 2022.
