Flexbox ist der gebräuchliche Name für das [CSS Flexible Box Layout Module](/de/docs/Web/CSS/Guides/Flexible_box_layout), ein Layout-Modell zum Anzeigen von Elementen in einer einzigen Dimension – als Reihe oder als Spalte.

In der Spezifikation wird Flexbox als ein Layout-Modell für die Benutzeroberflächengestaltung beschrieben. Das Hauptmerkmal von Flexbox ist, dass Elemente in einem Flex-Layout wachsen und schrumpfen können. Platz kann den Elementen selbst zugewiesen oder zwischen und um die Elemente herum verteilt werden.

Flexbox ermöglicht auch die Ausrichtung von Elementen auf der Haupt- oder Querachse und bietet so ein hohes Maß an Kontrolle über die Größe und Ausrichtung einer Gruppe von Elementen.
