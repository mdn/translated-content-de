`flex` ist ein Wert der CSS-{{cssxref("display")}}-Eigenschaft. Zusammen mit `inline-flex` führt es dazu, dass das Element, auf das es angewendet wird, zu einem {{Glossary("flex_container", "Flex-Container")}} wird, und die Kinder dieses Elements zu jeweils einem {{Glossary("flex_item", "Flex-Item")}} werden. Die Items nehmen dann am Flex-Layout teil, und alle Eigenschaften, die im [CSS-Flexbox-Layout-Modul](/de/docs/Web/CSS/CSS_flexible_box_layout) definiert sind, können angewendet werden.

Es gibt auch eine {{cssxref("flex")}}-Eigenschaft, die eine Kurzschreibweise für die Flexbox-Eigenschaften {{cssxref("flex-grow")}}, {{cssxref("flex-shrink")}} und {{cssxref("flex-basis")}} ist. Diese Eigenschaft ist nur auf Flex-Container anwendbar.

Zusätzlich kann [`<flex>`](/de/docs/Web/CSS/Reference/Values/flex_value) einen [flexiblen Abstand](/de/docs/Web/CSS/Reference/Values/flex_value) im CSS-Grid-Layout bezeichnen.
