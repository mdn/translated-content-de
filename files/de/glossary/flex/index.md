`flex` ist ein Wert der CSS-{{cssxref("display")}}-Eigenschaft. Zusammen mit `inline-flex` bewirkt es, dass das Element, auf das es angewendet wird, zu einem {{Glossary("flex_container", "Flex-Container")}} wird und die Kinder des Elements jeweils zu einem {{Glossary("flex_item", "Flex-Element")}} werden. Die Elemente nehmen dann am Flex-Layout teil, und alle Eigenschaften, die im [CSS Flexible Box Layout Modul](/de/docs/Web/CSS/Guides/Flexible_box_layout) definiert sind, können angewendet werden.

Es gibt auch eine {{cssxref("flex")}}-Eigenschaft, die eine Kurzform für die Flexbox-Eigenschaften {{cssxref("flex-grow")}}, {{cssxref("flex-shrink")}} und {{cssxref("flex-basis")}} ist. Diese Eigenschaft ist nur auf Flex-Container anwendbar.

Darüber hinaus kann [`<flex>`](/de/docs/Web/CSS/Reference/Values/flex_value) sich auf eine [flexible Länge](/de/docs/Web/CSS/Reference/Values/flex_value) im CSS-Grid-Layout beziehen.
