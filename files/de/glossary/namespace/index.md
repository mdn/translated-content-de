Ein Namespace ist ein Kontext für Bezeichner, eine logische Gruppierung von Namen, die in einem Programm verwendet werden. Innerhalb desselben Kontextes und desselben Geltungsbereichs muss ein Bezeichner eine Entität eindeutig identifizieren.

In einem Betriebssystem ist ein Verzeichnis ein Namespace. Jede Datei oder jedes Unterverzeichnis innerhalb eines Verzeichnisses hat einen eindeutigen Namen; derselbe Name kann mehrfach in Unterverzeichnissen verwendet werden.

In HTML, CSS und auf XML basierenden Sprachen ist ein Namespace der explizit deklarierte oder implizierte Dialekt, dem ein Element (oder Attribut) angehört.
