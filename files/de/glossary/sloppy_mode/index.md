{{Glossary("ECMAScript", "ECMAScript")}} 5 und später erlauben es Skripten, sich für einen neuen [Strikten Modus](/de/docs/Web/JavaScript/Reference/Strict_mode) zu entscheiden, der die Semantik von JavaScript auf verschiedene Weise verändert, um seine Widerstandsfähigkeit zu verbessern und Fehler verständlicher zu machen.

Der normale, nicht-strikte Modus von JavaScript wird manchmal als **Sloppy-Modus** bezeichnet. Dies ist keine offizielle Bezeichnung, aber Sie werden wahrscheinlich darauf stoßen, wenn Sie sich ernsthaft mit JavaScript-Code beschäftigen.
