**DNS** (Domain Name System) ist ein hierarchisches und dezentrales Namenssystem für internetverbundene Ressourcen. DNS führt eine Liste von {{Glossary("domain_name", "Domainnamen")}} zusammen mit den Ressourcen, wie z.B. IP-Adressen, die ihnen zugeordnet sind.

Die wichtigste Funktion von DNS ist die Übersetzung von benutzerfreundlichen Domainnamen (wie mozilla.org) in eine numerische {{Glossary("IP_address", "IP-Adresse")}} (wie 192.0.2.172); dieser Prozess der Zuordnung eines Domainnamens zur entsprechenden IP-Adresse wird als **DNS-Abfrage** bezeichnet. Im Gegensatz dazu wird eine **Reverse-DNS-Abfrage** (rDNS) verwendet, um den Domainnamen zu bestimmen, der mit einer IP-Adresse verknüpft ist.
