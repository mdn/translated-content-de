Ein **CSS-Deskriptor** definiert die Merkmale einer [At-Regel](/de/docs/Web/CSS/CSS_syntax/At-rules). At-Regeln können einen oder mehrere Deskriptoren haben. Jeder Deskriptor hat:

- Einen Namen
- Einen Wert, der die Komponentenwerte hält
- Ein "!important"-Flag, welches im Normalzustand nicht gesetzt ist
