**HTTPS** (**_HyperText Transfer Protocol Secure_**) ist eine verschlüsselte Version des {{Glossary("HTTP", "HTTP")}}-Protokolls. Es verwendet {{Glossary("TLS", "TLS")}}, um die gesamte Kommunikation zwischen einem Client und einem Server zu verschlüsseln. Diese sichere Verbindung erlaubt es den Clients, sensible Daten sicher mit einem Server auszutauschen, beispielsweise bei Bankgeschäften oder beim Online-Shopping.
