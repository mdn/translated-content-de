**Karten sortieren** ist eine einfache Technik, die in der {{Glossary("Information_architecture", "Informationsarchitektur")}} verwendet wird. Dabei werden Personen, die an der Gestaltung einer Website (oder eines anderen Produkttyps) beteiligt sind, eingeladen, die Inhalte / Dienste / Funktionen zu notieren, die ihrer Meinung nach in das Produkt aufgenommen werden sollten, und diese Funktionen dann in Kategorien oder Gruppen zu organisieren. Dies kann beispielsweise verwendet werden, um herauszufinden, was auf jeder Seite einer Website stehen sollte.
