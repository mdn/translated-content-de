**Scroll Snapping** ist eine Funktion, bei der der Inhalt "einrastet", wenn ein Scrollvorgang abgeschlossen ist, z.B. an der Position eines Absatzes, eines Satzes oder eines Bildrandes, anstatt an einem beliebigen Punkt zu stoppen.

Ein Beispiel: Ein Karussell ohne Scroll Snapping könnte das Scrollen mitten in einem Bild beenden, was dazu führt, dass dieses teilweise sichtbar bleibt.
Scroll Snapping bietet eine bessere Benutzererfahrung, indem es Entwicklern ermöglicht, sicherzustellen, dass ein vollständig sichtbares Bild an seine Position einrastet.

Scroll Snapping kann mit [CSS-Scroll-Snapping-Funktionen](/de/docs/Web/CSS/Guides/Scroll_snap) oder JavaScript implementiert werden.
