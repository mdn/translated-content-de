Ein **Block** auf einer Webseite ist ein {{Glossary("HTML", "HTML")}}-{{Glossary("element", "Element")}}, das in einem horizontalen Schreibmodus in einer neuen Zeile erscheint, d.h. unter dem vorhergehenden Element und über dem folgenden Element (allgemein bekannt als _Block-Level-Element_). Zum Beispiel ist {{htmlelement("p")}} standardmäßig ein Block-Level-Element, während {{htmlelement("a")}} ein _Inline-Element_ ist — Sie können mehrere Links nebeneinander in Ihrem HTML-Quellcode platzieren, und sie werden im gerenderten Ergebnis auf derselben Zeile angezeigt.

Mit der {{cssxref("display")}}-Eigenschaft können Sie ändern, ob ein Element inline oder als Block angezeigt wird (neben vielen anderen Optionen); **Blocks** unterliegen auch den Effekten von Positionierungsschemata und der Verwendung der {{cssxref("position")}}-Eigenschaft.
